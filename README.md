Meu primeiro projeto criado com base no template disponibilizado na aula de Imersão da Alura + AI com Google Gemini.

Neste projeto, alterei o modelo para servir como base de Softwares Homologados por uma empresa fictícia, alterando o layout, as cores e a forma como alguns objetos são apresentados.

Além disso, com ajuda do Google Gemini, adicionei uma funcionalidade que permite a inclusão de novas entradas para serem consultadas porteriormente.
